# 猜數字

[English](./README.md)

這是一個使用 Python 建構的簡單命令列遊戲。程式會根據使用者自訂的範圍產生一個隨機數字，玩家必須猜出這個數字是多少，遊戲會提示（「太高」或「太低」）來引導玩家找到正確答案。

建立這個專案是為了練習 Python 的流程控制（迴圈與條件判斷）、函式以及使用者輸入的處理技巧。

## 功能特色
- 隨機生成機制：每一局遊戲都會產生全新的隨機數字。
- 輸入驗證：防止使用者輸入非數字格式時導致程式崩潰。
- 追蹤次數：遊戲結束時，會顯示使用者花了幾次嘗試才猜中。
- 重玩功能：一局結束後，會詢問使用者是否要立即再玩一次。

## 如何執行
1. 確認 Python 已經安裝在你的電腦上。
   
   ```bash
   python3 -V
   ```

2. 下載或 Clone 此儲存庫。
   
   ```bash
   git clone ssh://git@codeberg.org/JurisByte/Guess.git
   ```

3. 開啟終端機或命令提示字元。
4. 進入專案資料夾。
   
   ```bash
   cd Guess/
   ```

5. 執行遊戲
   
   ```bash
   python3 main.py
   ```
